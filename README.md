# Домашнє завдання: Пошук зображень (TypeScript)

## Інструкція із запуску

Якщо ви хочете перевірити проєкт локально, переконайтеся, що ви встановили змінну `VITE_PIXABAY_API_KEY` (у файлі `.env` або у змінних середовища) перед запуском сервера розробки.

## Вимоги до HTML

- **Форма пошуку:** `.form` з інпутом, що має атрибут `name="search-text"`.
- **Контейнер галереї:** `.gallery` (у наданому шаблоні це `<ul>`). Розмітка галереї використовує посилання `<a href="largeImageURL">`, щоб бібліотека SimpleLightbox могла прив'язатися до селектора `.gallery a`.
- **Кнопка «Завантажити ще»:** `.load-more` (спочатку прихована).
- **Елемент завантажувача (лоадер):** `.loader` (керується функціями `showLoader`/`hideLoader`).

## Огляд реалізації

### `src/main.ts`
Оркестрація застосунку. Містить логіку взаємодії з DOM, обробники подій для форми пошуку (`.form`) та кнопки завантаження (`.load-more`), а також допоміжну функцію `fetchAndRender`, яка координує запити до API та оновлення інтерфейсу.

### `src/pixabay-api.ts`
HTTP-шар. Експортує функцію `getImagesByQuery(query, page)`, яка виконує запит до Pixabay за допомогою типізованого `axios.get<PixabayResponse>(...)` і повертає дані у форматі `PixabayResponse`.

### `src/render-functions.ts`
Допоміжні функції відображення. Цей файл експортує функцію `initRender`, яка приймає екземпляри DOM-елементів і повертає об'єкт з методами для керування UI: `createGallery`, `clearGallery`, `showLoader`/`hideLoader`, `showLoadMoreButton`/`hideLoadMoreButton` та `showToast`. Модуль також ініціалізує `SimpleLightbox` і викликає `lightbox.refresh()` після оновлення DOM.

### `src/pagination.ts`
Інкапсулює стан пагінації. Визначає клас `Pagination` та експортує константу `PER_PAGE` (за замовчуванням 15). Цей клас використовується для перемикання сторінок, скидання лічильника та перевірки, чи досягнуто кінця списку результатів.

### `src/types/`
Визначення типів TypeScript.
- `types/pixabay.ts`: містить інтерфейси `PixabayImage` та `PixabayResponse`, які використовуються в API-шарі.
- `types/simplelightbox.d.ts`: містить мінімальні оголошення типів для бібліотеки SimpleLightbox.

## Важливі рішення в реалізації

- **Впровадження залежностей (Dependency Injection) для рендерингу:** Модуль `render-functions.ts` не шукає елементи в DOM самостійно під час завантаження. Замість цього, функція `initRender({ gallery, loader, loadMoreButton })` викликається з `main.ts` і повертає вже налаштовані функції UI. Це покращує тестування та дозволяє уникнути побічних ефектів.
